JDK

Java Development Kit（JDK）是昇陽電腦針對Java開發人員發布的免費軟體開發套件（SDK，Software development kit）。自從Java推出以來，JDK已經成為使用最廣泛的Java SDK。由於JDK的一部分特性採用商業許可證，而非開源[1]。因此，2006年昇陽電腦宣布將發布基於GPL的開源JDK，使JDK成為自由軟體。在去掉了少量閉源特性之後，昇陽電腦最終促成了GPL的OpenJDK的發布。
